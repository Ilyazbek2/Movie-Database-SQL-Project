# Movie-Database-SQL-Project#

This project demonstrates SQL querying skills using a relational movie database.

## Skills Covered
- INNER JOINs
- GROUP BY and HAVING
- Subqueries
- Aggregation functions
- Real-world analytical queries

## Database Tables
- Movies
- Directors
- Actors
- Movie_Cast
- BoxOffice

## How to Run
1. Run `schema.sql` to create tables
2. Run `sample_data.sql` to insert data
3. Open `problems.sql` to view tasks
4. Check `solutions.sql` for answers

This project is suitable for:
- SQL portfolios
- Interview preparation
- University assignments
